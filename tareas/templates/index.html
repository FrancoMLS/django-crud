Django CRUD

Este sistema implementa un CRUD (Create, Read, Update, Delete) para gestionar tareas utilizando Django, un framework de desarrollo web en Python. Con esta aplicación, los usuarios pueden crear nuevas tareas especificando detalles como el título, la descripción y si dicha tarea es importante o no. Además, se pueden ver todas las tareas existentes en una lista, actualizar información de una tarea y eliminar tareas que ya no son necesarias. Todo esto se logra mediante una interfaz intuitiva y fácil de usar, proporcionando una experiencia fluida para la gestión eficiente de tareas.
